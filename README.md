# متجر الملابس الإلكتروني

متجر إلكتروني لبيع الملابس مع لوحة تحكم إدارية.

## المميزات

- عرض المنتجات مع المقاسات المختلفة
- نظام سلة تسوق متكامل
- طرق دفع متعددة (فودافون كاش، الدفع عند الاستلام)
- لوحة تحكم إدارية
- نظام إشعارات للعملاء
- وضع ليلي/نهاري

## متطلبات التثبيت

1. Node.js 16.x أو أعلى
2. MongoDB
3. حساب Railway للنشر

## كيفية التشغيل محلياً

### تشغيل الخادم

```bash
cd server
npm install
npm run dev
